# Student-Placement-Management-System
# 🎓 Student Placement Management System

A clean and simple Java console application that simulates the core functionalities of a student placement system — from student and company registration to applying, shortlisting, and final placement.

---

##  Features

✅ Register Students  
✅ Register Companies  
✅ View Eligible Companies (based on CGPA)  
✅ Apply for Companies  
✅ View Applications  
✅ Select Students  
✅ Mark Students as Placed  
✅ View All Students  
✅ View All Companies  

---

##  Technologies Used

- Java (JDK 17+)
- Object-Oriented Programming (OOP)
- ArrayList, HashMap
- Clean Console UI (Scanner-based input)
- Modular Design with Packages (`model`, `service`, `util`)

---

##  How to Run

1. Open terminal in project root
2. Compile:
javac model/.java service/.java Main.java
3. Run:
java Main
